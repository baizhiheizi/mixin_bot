Blaze WebSocket helpers. `blaze` opens a `Faye::WebSocket::Client` with `Bearer` auth, headers `User-Agent: mixin_bot/<VERSION>`, and `ping: 60`. `start_blaze_connect(reconnect: true, &block)` yields the socket; if `on_open`/`on_message`/`on_error`/`on_close` are not defined it logs via `p` and auto-acknowledges incoming messages with `ACKNOWLEDGE_MESSAGE_RECEIPT`. On close without reconnect, the socket is cleared. Send helpers: `blaze_send_plain_text`, `blaze_send_recall_message`, `blaze_send_post` (alias of plain text), `blaze_send_contact`, `blaze_send_app_card`, `blaze_send_app_button`, `blaze_send_group_app_button`. Categories: `PLAIN_TEXT`, `MESSAGE_RECALL`, `PLAIN_CONTACT`, `APP_CARD`, `APP_BUTTON_GROUP`. All payloads base64-url encoded (no padding) via `Base64.urlsafe_encode64`.
