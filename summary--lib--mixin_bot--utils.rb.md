# lib/mixin_bot/utils.rb

`MixinBot::Utils` extends four sub-modules:
- Address (main address, ghost keys, validation)
- Crypto (access_token JWT signing, generate_ed25519_key, encrypt_pin, decode_key, unique_uuid)
- Decoder (int decoding, key decoding, transaction decoding)
- Encoder (int encoding, key encoding, transaction encoding, binary packing)
